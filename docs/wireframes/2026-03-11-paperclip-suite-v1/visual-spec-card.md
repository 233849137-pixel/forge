# Visual Spec Card

## Main Reference

- Reference repo: `https://github.com/paperclipai/paperclip`
- Reference site: `https://www.trypaperclip.com/`
- Page family in scope:
  - 仪表盘
  - 项目管理
  - AI员工区（多子页）
  - 资产管理

## Must-Keep Traits

1. 大标题主导的叙事感
2. 深色背景上的杂志式 section 编排
3. 信息卡像“展板”一样分区陈列
4. 少量高亮色承担状态节奏
5. AI员工区拆成多个二级页，不做单页堆叠
6. 页面适合展示，不是传统浅色 admin

## Token Direction

- Background: `#09090b`
- Surface 1: `#121216`
- Surface 2: `#18181d`
- Paper: `#f3ede3`

- Text secondary: `#b4ac9f`

- Cobalt: `#84a8ff`

- Amber: `#ffd166`

## Font Direction

- Display: `Newsreader`
- Body: `IBM Plex Sans`
- Mono: `IBM Plex Mono`

## Layout Principles

1. 固定左侧导航 + 右侧滚动工作区
2. 每页首屏必须有英雄区和一句价值表达
3. section 之间使用大留白，不做密集表格堆叠
4. 卡片圆角偏大，边框低对比，阴影更像展台灯光
5. 移动端优先塌陷成单列，不保留复杂多栏

## Scope Freeze

- 只做新 wireframe，不改既有页面实现
- 只做静态 mock，不接真实数据
- AI员工区按 `团队总览 / 组织架构 / 目标对齐 / 工单中心 / 心跳排班 / 成本控制 / 治理控制 / 员工详情` 拆页
- 文案统一中文优先
